# 대화 요약 Skill

사용자가 제공한 대화 내용을 분석하여 요약합니다.

## 사용 방법

### 기본 사용
```
/summarize
(대화 내용 붙여넣기)
```

### 특정 주제로 요약
```
/summarize 테스트 전략
(대화 내용 붙여넣기)
```

## 입력

사용자가 다음 중 하나를 제공합니다:
- **AI와의 대화**: 이전 AI 세션의 대화 내용
- **피어와의 대화**: Slack, Discord, 대면 논의 내용 등

## 요약 주제 (선택)

$ARGUMENTS

주제가 지정되면 해당 관점에서 요약합니다:
- `테스트 전략` → 테스트 관련 결정사항 중심
- `아키텍처` → 설계 결정 중심
- `구현 방향` → 코드 작성 방향 중심
- (미지정) → 전체 내용 요약

## 출력 형식

```markdown
## 대화 요약

### 요약 주제
(지정된 주제 또는 "전체")

### 핵심 내용
- 포인트 1
- 포인트 2
- 포인트 3

### 결정사항
| 항목 | 결정 | 이유 |
|------|------|------|
| ... | ... | ... |

### 미결정/추가 논의 필요
- 항목 1
- 항목 2

### 다음 할 일
- [ ] 액션 1
- [ ] 액션 2
```

## 예시

### 예시 1: 기본 요약
```
/summarize

[대화 내용]
A: 테스트 어떻게 할까?
B: 단위 테스트부터 시작하자
A: Option.decrease()가 제일 중요한 것 같아
B: 동의해. 재고 관련이니까
```

### 예시 2: 주제 지정
```
/summarize 테스트 우선순위

[대화 내용]
...
```
→ 테스트 우선순위 관련 내용만 추출하여 요약
